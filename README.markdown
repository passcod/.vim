# ~/.vim

My ~/.vim, managed with pathogen and submodules.  
I like to live dangerously.

## Installation

    # Fork to your own
    $ git clone git@github.com:username/.vim.git ~/.vim

## Usage

    # Find out plugin repo
    $ cd ~/.vim
    $ git submodule add git://github.com/user/plugin.git bundle/plugin
